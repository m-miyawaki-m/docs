### 小論文
chatGPTを使って小論文の採点を行う。


```prompt
あなたは小論文の採点官です。小論文を「論理性」「構成」「表現力」の3つの観点から100点満点で採点してください。
また、良かった点と改善すべき点をフィードバックしてください。

採点結果は以下のフォーマットで返してください：
- 総合評価: XX点 / 100点
- 評価：
    - 論理性: XX点 / 35点
    - 構成: XX点 / 35点
    - 表現力: XX点 / 30点
- フィードバック:
    - 良かった点:
    - 改善すべき点:
```

```prompt

【テーマ】
（ここにてテーマを貼り付け）

【問題】
（ここに問題文を貼り付け）

【解答】
（ここに解答を貼り付け）

```


```prompt

【テーマ】
「序論・本論・結論（SBC）」の型を意識して、次のテーマで小論文を書く。（400文字）

【問題】
「AI技術の発展と社会への影響」


【解答】
序論
AI技術は現在、検索の結果を最適化するため等に利用がされている。


本論
例えば、検索エンジンの結果に不適切な結果が含まれないように仕分けする等だ。
メリットは、人が仕分けするよりも膨大な量を短時間で処理できる。


結論
今後、検索結果の最適化等に留まらず様々なデータ処理に活用されていくだろう。
そのため、AI技術に対して正確な知識を取得する等、準備を行う事でAI技術の発展を上手に付き合っていく必要がある


```

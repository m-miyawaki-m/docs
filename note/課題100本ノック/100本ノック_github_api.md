## **使用ライブラリ**
✅ **Python の `requests` ライブラリ**
```python
import requests

token = "your_github_token"
headers = {"Authorization": f"Bearer {token}"}

url = "https://api.github.com/user"
response = requests.get(url, headers=headers)
print(response.json())
```

✅ **PyGithub ライブラリ**
```python
from github import Github

token = "your_github_token"
g = Github(token)

user = g.get_user()
print(user.login)
```
